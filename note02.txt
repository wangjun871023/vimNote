1 NERDTree插件 
1.1 安装插件
    Vundle Plugin 'scrooloose/nerdtree'
1.2 插件功能
    目录树：快速打开文件，文件的使用等等
1.3 常用命令
    :NERDTree  打开/更新目录树 
    :NERDTreeToggle 打开/关闭目录树 
    :NERDTreeMirror 镜像目录树到当前的tab页
    :NERDTreeClose  关闭目录树
1.4 NERDTree的使用命令
    1. 文件节点操作  
       双击/回车 :打开相应的文件
       o : 打开文件进行显示
       go: 预览指定文件
       t : 在新的tab页中进行打开文件
       T : 在新的tab页中进行打开文件,不进行跳转当前页面
       i : 以上下/水平分窗口进行打开文件
       gi: 以上下/水平分窗口进行预览显示文件
       s : 以左右/垂直分窗口进行打开文件
       gs: 以左右/垂直分窗口进行预览显示文件
    2. 目录节点操作
       o :打开/关闭目录节点
       O :递归打开目录节点
       x :关闭当前目录节点
       X :递归关闭当前打开的所有的目录节点
    3. 目录插件操作
       q : 关闭目录插件
       A : 最大化/最小化目录窗口
       ? : 打开/关闭帮助  
    4. 文件系统操作
       C : 改变目录目录树根目录选择路径 
       u : 上一层目录
       U : 上一层目录,并且当前保持打开目录
       r : 刷新当前目录
       R : 刷新树目录
       m : 显示菜单
    5. 目录树导航操作
       P : 返回当前目录树的根节点
       p : 返回至目录的父节点
       K : 到第一个子节点
       J : 到最后一个子节点
       <C-j>: 到再一个兄弟节点
       <C-k>: 到上一个兄弟节点
    6. 目录过滤操作
       I : 显示/隐藏 隐藏文件(.*)
       F : 显示/隐藏 文件
       B : 显示/隐藏 书签 
    7. 书签操作
       :Bookmark name  创建一个name的书签
       :OpenBookmark name  打开name的书签
       :ClearBookmarks names  清除指定的书签或是全部书签
       o: 打开书签
       t: 在一个新的tab页中进行打开书签
       T: 在一个新的tab页中进行打开书签,保持在当前页，不跳转
       D: 删除指定书签
