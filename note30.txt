1 vim-signify 插件
1.1 安装
Plugin 'mhinz/vim-signify'
1.2 简介
    实时显示修改状态，可以同时支持git,svn,cvs等主流的版本管理工具。
1.3 git常用命令
    git clone: 从指定的仓库中同步项目，软件。
        git clone https:// .....
    git init : 初始化本地仓库
    git status : 查看仓库的状态
    git add *.txt : 将*.txt添加到本地暂存区
    git commit -m '第一次提交' : 提交到本地仓库
    git remote add origin https://github.com/wangjun871023/vimNote.git : 远程仓库与本地仓库关联
    git push -u origin master    : 将本地仓库上传到github上
1.4 使用
    + ：  添加行
    _n ： 删除行
    ！ ： 此行有修改
