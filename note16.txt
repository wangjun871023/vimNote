1 ctrlp-funky 插件
1.1 安装
Plugin 'tacahiroy/ctrlp-funky'
1.2 使用
    ctrlp-funky功能：模糊搜索当前文件中的所有函数,对CtrlP插件的扩展。
    :CtrlPFunky ：激活插件功能
1.3 例子
public class Hello {
	public static void main(String[] args){
		System.out.println("Hello world!");
	}
	public static add(int a,int b){
		return a+b;
	}
	public static sum(int a,int b){
		return a+b;
	}
}
